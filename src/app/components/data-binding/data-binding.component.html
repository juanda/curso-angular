Data binding

Desde el modelo hacia el DOM

Se puede cambiar un contenido usando con el valor del modelo usando interpolación:

Ejemplo interpolación:

El número es "{{ modelo.numero }}" la cadena es "{{ modelo.cadena}}" )

El número es "{{ modelo.numero }}" y la cadena es "{{ modelo.cadena}}"

O se puede cambiar:

  • Una propiedad
  • Un estilo
  • Una clase
  • Un atributo

Ejemplo propiedad:

<input [value]="modelo.propiedad" />

Ejemplo estilo

<span [style.color]="modelo.estilo">Color rojo</span>

Color rojo

Ejemplo de clase

<button [class]="modelo.clase">Botón</button>

Desde el DOM hacia el modelo

Se hace a través de eventos

La tecla pulsada es: {{modelo.tecla}}

La cadena ahora vale: {{ modelo.cadena }}

Del modelo al DOM y del DOM al modelo simultaneamente(two-way data binding)

Forma "enrevesada", usando [] y () simultaneamente

<input [value]="modelo.localidad" (keyup)="showLocalidad($event);"/>

Valor localidad: {{ modelo.localidad }}

Forma adecuada, mediante la directiva NgModel

<input [(ngModel)]="modelo.localidad"/>

Valor localidad: {{ modelo.localidad }}
